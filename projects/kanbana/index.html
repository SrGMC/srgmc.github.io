Kanbana allows tracking users and projects from flat markdown files using Kanban boards.

Kanbana permite crear Kanban boards para hacer seguimiento de usuarios y proyectos a partir de archivos Markdown

Github

Kanbana is based of the Crepido project by @issmirnov.

Kanbana está basado en el proyecto Crepido de @issmirnov.

Kanbana is easy to use and uses flat Markdown files that every text editor can open

Kanbana es fácil de usar y usa archivos Markdown que todo editor de texto puede abrir

It has the following features:

Implementa las siguientes funcionalidades:

  • Dark and light mode
  • Markdown formatting
  • Links
  • Labels and custom labels
  • Timers
  • Tasks
  • Multiple boards in the same page
  • Images served from the server
  • Responsive design
  • Integrated caching
  • Modo oscuro y claro
  • Formato markdown
  • Enlaces
  • Etiquetas y etiquetas personalizadas
  • Timers
  • Tareas
  • Múltiples boards en una misma página
  • Imagenes servidas desde el servidor
  • Diseño responsive
  • Caché integrada
