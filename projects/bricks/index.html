Bricks is a Webapp that allows to simulate the execution of code in the MIPS32 architecture. It is inspired by QtSpim with an interface focused mainly on iOS.

Bricks es una Webapp que permite simular la ejecución de código en la arquitectura MIPS32. Está inspirado por QtSpim con una interfaz enfocada principalmente en iOS.
