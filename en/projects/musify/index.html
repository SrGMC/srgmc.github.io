Musify is a simple and fast music player, built from the ground up, using web technologies such as HTML, CSS and JavaScript, all wrapped on Electron Framework, available for macOS.

GitHub

Note: This application is not currently maintained. It is possible that I will take it up again in the future.

Features

Musify is a simple music player that features:

  • Dark and light modes
  • Loop mode
  • Notifications about song change

This is my first project as a computer science student. I've tried my best to write a code that follows common coding conventions, but keep in mind that you may find common beginner errors. If so, feel free to help and contribute to improve this app.
