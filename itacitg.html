Is there a car in the garage?

Inicio CV Proyectos
Is there a car in the garage? es un proyecto de Arduino que utiliza un sensor ultrasónico para saber si hay un coche en el garaje. Accesible a traves de una interfaz web simple.
Descargar código
Apple Shortcut
GitHub

It is originaly programmed to detect a car in a closed space, but it can be used to detect any static object in a closed space.

After downloading the code, check out installation instructions.

Once installed, check out usage instructions. TL;DR: It's easy!. Download the Apple Shortcut on an iOS device (iOS 13 and up), configure with URL, username and password and you're done!. Ask Siri about it next time.
